# Metam

Metam is a gem that extends a Model with dynamic attributes and validators during runtime.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'metam'
```

And then execute:

```ruby
$ bundle
```

<!-- Or install it yourself as:
```ruby
$ gem install metam
``` -->
